## dataset note

the original FIR dataset used for training and ward-level crime analysis was too large for github upload limits (>500MB), so it is excluded from this repo.  

the project still includes:
- ward boundary data
- BBMP infrastructure datasets
- full app logic
- spatial risk modelling
- ensemble scoring pipeline

to run locally, place the FIR dataset inside:

data/FIR_Details_Data.csv
